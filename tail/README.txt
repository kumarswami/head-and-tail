	In this project the options like -n, -c, -q, -v, -f,options are done. 

Option -n, Output the last K lines, instead of the default of the last 10;
alternatively, use "-n +K" to output lines starting with the Kth.

	$ ./project -n 12 file.txt

Option -c, --bytes=K 	Output the last K bytes; alternatively, use "-c +K" to output bytes starting with the Kth byte of each file.

	$ ./project -c 4 file.txt
	
Option -q, 	Never output headers giving file names.

	$ ./project -q file.txt
	
Option -v, Always output headers giving file names.

	$ ./project -v file.txt	

Option -f, Output appended data as the file. If name is specified, the file with filename.

The --help option will display the uses, synopsis and the examples for how to use the specified options for execution of the command.
After using the syntax specified in --help file it gives appropriate output.
